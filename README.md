# 📊 Data Analytics & AI Bootcamp – Portfolio (ADALAB)

Este repositorio recoge los ejercicios, proyectos y prácticas que he desarrollado durante mi formación en el **Bootcamp de Data Analytics & Inteligencia Artificial** de [ADALAB](https://adalab.es).  
El objetivo es demostrar mis competencias en el manejo del ciclo completo del dato: extracción, transformación, análisis, visualización y comunicación de insights, con un marcado enfoque en herramientas de IA aplicada.

## 🐍 Python – Ejercicios y proyectos

### 1. Lógica de programación con juegos clásicos
- **[Descripción del ejercicio]** – Ejemplo: *PTrivia, Tic-Tac-Toe*. Se trabajaron condicionales, bucles, entrada de usuario y aleatoriedad.  
- *Tecnologías:* Python estándar.

### 2. Extracción de datos: APIs y web scraping
- **[Descripción del ejercicio]** – Ejemplo: *MusicStream* – obtención de datos de artistas y rankings musicales desde Deezer API, Last.fm API y scraping de sitios de crítica. Almacenamiento en MySQL.  
- *Tecnologías:* `requests`, `Beautiful Soup`, `Selenium`, `MySQL connector`.

### 3. Manipulación y análisis con Pandas / NumPy
- **[Descripción del ejercicio]** – Limpieza de datasets reales (valores nulos, duplicados, outliers), análisis exploratorio (EDA), agregaciones y merges.  
- *Tecnologías:* `pandas`, `numpy`, `matplotlib`.

### 4. Visualización de datos con Matplotlib y Seaborn
- **[Descripción del ejercicio]** – Creación de gráficos avanzados: histogramas, boxplots, heatmaps, pairplots, y personalización de estilos.  
- *Tecnologías:* `matplotlib`, `seaborn`.

### 5. Estadística y pruebas A/B
- **[Descripción del ejercicio]** – Ejemplo: *Optimización de talento* – análisis de satisfacción y retención de empleados, prueba A/B con `scipy.stats` (t-test, chi-cuadrado).  
- *Tecnologías:* `scipy`, `statsmodels`, `pandas`.

### 6. ETL y automatización con Apache Airflow
- **[Descripción del ejercicio]** – Diseño de DAGs para orquestar tareas de extracción, transformación y carga (ETL) con ejecución programada.  
- *Tecnologías:* `Apache Airflow`, `Python`, `MySQL`.
